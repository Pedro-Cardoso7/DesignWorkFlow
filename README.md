# MJ Designer Workflow — Chrome Extension

Streamline your Midjourney outfit crop and organize workflow.

---

## Installing the Extension

### Step 1 — Get the `dist` folder

Pedro will send you a `dist` folder (or a ZIP file containing it). If you received a ZIP, extract it first — you should end up with a folder called `dist`.

### Step 2 — Open Chrome Extensions

1. Open **Google Chrome**
2. In the address bar, type `chrome://extensions` and press **Enter**

### Step 3 — Enable Developer Mode

In the top-right corner of the Extensions page, toggle **Developer mode** ON.

![Developer mode toggle in top-right corner]

### Step 4 — Load the Extension

1. Click **"Load unpacked"** (appears after enabling Developer mode)
2. In the file picker, navigate to and select the `dist` folder
3. Click **Select Folder**

The extension will appear in your extensions list immediately.

### Step 5 — Pin it to the toolbar (optional but recommended)

1. Click the puzzle piece icon (🧩) in the Chrome toolbar
2. Find **MJ Designer Workflow** and click the pin icon next to it

---

## Using the Extension

1. Go to [midjourney.com](https://www.midjourney.com)
2. Click the **MJ Designer Workflow** icon in your toolbar to open the side panel
3. The panel opens on the right side of the browser

---

## Updating the Extension

When Pedro sends you a new `dist` folder:

1. Replace your old `dist` folder with the new one (keep it in the same location)
2. Go to `chrome://extensions`
3. Find MJ Designer Workflow and click the **refresh icon** (↺)

---

## Troubleshooting

**Extension not showing up after loading:**
Make sure you selected the `dist` folder itself, not the folder containing it.

**Side panel not opening:**
Make sure you're on `midjourney.com`. The extension only works on that site.

**Something looks broken after an update:**
Go to `chrome://extensions`, click the refresh icon on the extension, then hard-refresh Midjourney (`Ctrl + Shift + R`).
